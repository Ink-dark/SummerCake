# SummerCake Git GUI

SummerCake 是一个基于 Qt 6.10.0 Widgets 模块开发的 Git GUI 中文版，采用动态链接 Qt 库的方式构建，并原生集成AI助手功能，旨在帮助用户更方便地管理Git仓库。

## 功能特性

### 1. 仓库管理
- 打开本地仓库
- 克隆远程仓库
- 初始化新仓库

### 2. 工作区管理
- 显示文件状态（已修改、已暂存、未跟踪等）
- 文件差异对比
- 暂存/取消暂存文件
- 丢弃修改

### 3. 提交管理
- 提交修改
- 查看提交历史
- 查看提交详情
- 智能生成提交信息（AI功能）

### 4. 分支管理
- 查看分支列表
- 创建、切换、合并、删除分支

### 5. 远程操作
- 推送代码
- 拉取代码
- 查看远程仓库信息

### 6. AI助手功能
- 智能生成提交信息
- 代码审查建议
- 冲突解决助手
- Git命令解释
- 仓库健康分析
- AI助手悬浮窗
- 支持多种AI服务提供商（OpenAI、本地模型等）
- 隐私模式（仅使用本地模型）

## 技术架构

### 开发环境
- Qt 6.10.0
- C++17
- CMake

### 项目结构
```
SummerCake/
├── src/                      # 源代码
│   ├── git/                  # Git操作核心
│   ├── ai/                   # AI功能模块
│   ├── widgets/              # UI组件
│   └── main.cpp              # 程序入口
├── resources/                # 资源文件
│   ├── icons/               # 图标
│   └── translations/        # 翻译文件
├── CMakeLists.txt           # CMake构建文件
└── README.md               # 项目说明文档
```

### 核心类设计

#### GitManager
- 封装Git命令调用
- 处理Git仓库操作
- 提供文件状态、提交历史、分支管理等功能

#### AIManager
- 管理AI服务提供商
- 处理AI请求和响应
- 支持多种AI服务
- 实现隐私模式

#### AIProvider
- AI服务提供商接口
- 支持OpenAI、本地模型等多种实现

## 安装指南

### 前置条件
- Qt 6.10.0
- Git
- CMake 3.16+（需要安装并添加到系统PATH）

### 构建步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/Ink-dark/SummerCake.git
   cd SummerCake
   ```

2. **创建构建目录**
   ```bash
   mkdir build
   cd build
   ```

3. **配置CMake**
   ```bash
   cmake ..
   ```

4. **构建项目**
   ```bash
   cmake --build . --config Release
   ```

5. **部署依赖**
   ```bash
   windeployqt --release bin/SummerCake.exe
   ```

### 运行程序

```bash
./bin/SummerCake.exe
```

## 使用说明

### 1. 打开仓库

- 点击工具栏的"打开仓库"按钮
- 选择包含.git目录的文件夹
- 仓库将在主窗口中打开

### 2. 查看文件状态

- 在"文件状态"标签页中查看所有文件的状态
- 右键点击文件可以进行暂存、取消暂存、丢弃修改等操作

### 3. 提交修改

- 暂存需要提交的文件
- 点击工具栏的"提交"按钮
- 输入提交信息（可以使用AI生成建议）
- 点击"确定"完成提交

### 4. 分支管理

- 在左侧"分支"标签页中查看所有分支
- 右键点击分支可以进行切换、合并、删除等操作

### 5. 使用AI助手

- 点击工具栏的"AI助手悬浮窗"按钮打开AI助手
- 在输入框中输入问题，如"解释git commit命令"
- 点击"发送"按钮获取AI响应

### 6. 配置AI设置

- 点击工具栏的"AI配置"按钮
- 选择AI服务提供商
- 配置API密钥等参数
- 启用/禁用AI功能
- 启用/禁用隐私模式

## 界面预览

### 主窗口

- **菜单栏**：包含文件、编辑、仓库、分支、远程、AI助手、帮助等菜单
- **工具栏**：常用操作按钮
- **左侧面板**：仓库导航树、分支列表、标签列表
- **中间面板**：文件状态列表、提交历史
- **右侧面板**：文件差异对比、提交详情、AI建议
- **底部状态栏**：当前分支、仓库状态、AI状态

### AI助手悬浮窗

- 可拖动的悬浮窗口
- 显示用户查询和AI响应
- 支持发送新查询
- 可随时关闭

## 隐私说明

- 当启用隐私模式时，所有AI请求将仅使用本地模型，不会上传任何代码或数据
- 当使用云端AI服务时，会发送必要的数据到AI服务提供商
- 用户可以随时在设置中调整AI功能和隐私选项

AI功能采用插件架构设计，便于扩展新的AI服务提供商。
